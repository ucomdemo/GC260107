## MyVueTodo

待辦事項管理系統 - 使用 Vue 3 + Pinia + Bootstrap 構建的響應式待辦清單應用。

### 功能特色
- ✅ 響應式網頁設計，支援桌面和移動設備
- ✅ 使用 Bootstrap 5 實現美觀的 UI 介面
- ✅ 使用 Pinia 進行狀態管理
- ✅ 使用 Vue Router 實現頁面路由
- ✅ 完整的待辦事項 CRUD 功能
  - 新增待辦事項
  - 編輯待辦事項（inline 編輯）
  - 刪除待辦事項（含確認對話框）
  - 標記完成/未完成
  - 顯示統計資訊（總計、已完成、未完成）
- ✅ 版權宣告頁面

### 技術棧
- Vue 3 (Composition API)
- Vite
- Bootstrap 5
- Bootstrap Icons
- Pinia (狀態管理)
- Vue Router (路由管理)

### 使用方式
- 安裝相依套件：`npm install`
- 啟動開發伺服器：`npm run dev`
- 建置產出：`npm run build`
- 預覽產出：`npm run preview`
